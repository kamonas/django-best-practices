Django ベストプラクティス
================================

.. image:: http://i.creativecommons.org/l/by-nc-sa/3.0/88x31.png
    :alt: Creative Commons Attribution-NonCommercial-ShareAlike 3.0 Unported

このドキュメントは `Django Best Practices <http://lincolnloop.com/django-best-practices>`_ の日本語版です。
原著者は  `Brian Rosner <http://oebfare.com/>`__ , `Eric Holscher <http://ericholscher.com/>`__ です。

`(翻訳版 github リポジトリ) <https://github.com/peketamin/django-best-practices>`_

----

これは `Django ウェブフレームワーク <http://www.djangoproject.com>`_ の開発とデプロイにおけるベストプラクティスとして、随時更新されるドキュメントです。これらは Django で作業する上での、 *正しい* やり方、 *唯一の* やり方を指南するものではありません。その代わり、我々が何年にも渡りこのフレームワークを使い磨いてきたベストプラクティスではあります。

そしてこれは、 `Brian Rosner <http://oebfare.com>`__ と `Eric Holscher <http://ericholscher.com>`__ によって、再利用可能な Django アプリケーションを書き、保守するためのベストプラクティスにとして始められた、偉大な `django-reusable-app-docs project <http://github.com/ericholscher/django-reusable-app-docs>`__ のフォークです。

.. note:: このドキュメントのソースコードは GitHub 上の `django-best-practices <http://github.com/lincolnloop/django-best-practices>`__ (オリジナル) にあり、 `Sphinx <http://sphinx.pocoo.org>`__ を使うことでいくつものフォーマットでビルド可能です。
