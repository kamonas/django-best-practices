Django ベストプラクティス
================================


これは `Django ウェブフレームワーク <http://www.djangoproject.com>`__ の開発とデプロイにおけるベストプラクティスとして、随時更新されるドキュメントです。これらは Django で作業する上での、*正しい* やりかた、*唯一の* やり方を指南するものではありません。その代わり、我々が何年にも渡りこのフレームワークを使い磨いてきたベストプラクティスではあります。

そしてこれは、`Brian Rosner <http://oebfare.com>`__ と `Eric Holscher <http://ericholscher.com>`_ によって、再利用可能な Django アプリケーションを書き、保守するためのベストプラクティスにとして始められた、偉大な `django-reusable-app-docs project <http://github.com/ericholscher/django-reusable-app-docs>`__ のフォークです。

.. note:: このドキュメントのソースコードは GitHub 上の `django-best-practices <http://github.com/lincolnloop/django-best-practices>`__ にあり、 `Sphinx <http://sphinx.pocoo.org>`__ を使うことでいくつものフォーマットでビルド可能です。
